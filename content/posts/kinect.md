# Testing functions using microsoft kinect
## What is microsoft kinect

Microsoft Kinect is a motion-sensing input device initially developed for the Xbox 360 gaming console and later integrated with Windows PCs. Released in 2010, it was designed to enable users to control and interact with their Xbox games or applications without the need for a traditional controller. Kinect uses a combination of depth-sensing cameras, an RGB camera, and an array of microphones to detect motion, recognize faces, and process voice commands.

Key Components:
- RGB Camera: Captures standard color images for video, photos, and facial recognition.
- Depth Sensor: Uses infrared technology to create a 3D map of the environment, allowing Kinect to track full-body motion.
- Microphone Array: A set of microphones for voice recognition, enabling users to give voice commands.
Tilt Motor: Adjusts the angle of the device to better capture user mov

These components can be used for many applications, such as movement recognition, 3D recognition, segmentations and so on. We tried some of these features during the lab.
